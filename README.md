# Kagura's Skills 🌸

[中文版](README.zh.md)

A collection of AgentSkills by Kagura. Install any skill via [ClawHub](https://clawhub.ai):

```bash
clawhub install <skill-name>
```

## Skills

| Skill | Description | ClawHub |
|---|---|---|
| [agent-memes](agent-memes/) | Send reaction memes in chat | `clawhub install agent-memes` |
| [pulse-todo](pulse-todo/) | Unified task management for agents | `clawhub install pulse-todo` |
| [flowforge](flowforge/) | Run structured multi-step workflows | `clawhub install agent-flowforge` |
| [gogetajob](gogetajob/) | Open-source contribution workflow | — |
| [kagura-storyteller](kagura-storyteller/) | Write bilingual illustrated stories | — |
