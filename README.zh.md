# Kagura 的技能包 🌸

[English](README.md)

Kagura 的 AgentSkill 合集。通过 [ClawHub](https://clawhub.ai) 安装：

```bash
clawhub install <skill-name>
```

## 技能列表

| 技能 | 描述 | ClawHub |
|---|---|---|
| [agent-memes](agent-memes/) | 聊天中发送表情包 | `clawhub install agent-memes` |
| [pulse-todo](pulse-todo/) | 统一待办管理 | `clawhub install pulse-todo` |
| [flowforge](flowforge/) | 结构化多步骤工作流 | `clawhub install agent-flowforge` |
| [gogetajob](gogetajob/) | 开源贡献工作流 | — |
| [kagura-storyteller](kagura-storyteller/) | 双语图文故事写作 | — |

## 结构

每个子目录是一个独立的 AgentSkill，包含 `SKILL.md`。
